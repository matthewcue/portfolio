# Contributing

Thanks for helping improve this portfolio!

## Local setup

```bash
npm install
npm run dev
```

## Content updates

- Projects: `src/content/projects.ts`
- Labs: `src/content/labEntries.ts`
- Skills + roadmap: `src/content/skills.ts`
- Writing: `src/content/posts.ts`
- Profile: `src/content/profile.ts`

Keep entries short and consistent with the existing examples.

## Deploying

Pushes to `main` trigger GitHub Actions to build and deploy the site to GitHub Pages.
